# 如何找到博客來分類頁面的正確 URL

## 目前狀態

目前爬蟲程式**主要使用搜尋功能**來收集書籍，這是最穩定可靠的方法。分類瀏覽功能是選用的，只有在您提供正確的分類 URL 時才會使用。

## 為什麼需要正確的 URL？

博客來網站的分類頁面 URL 格式可能會變動，而且不同分類的 URL 結構可能不同。如果您想使用分類瀏覽功能來收集特定分類的書籍，需要提供正確的完整 URL。

## 如何找到正確的分類 URL

### 方法一：從博客來網站直接複製

1. **開啟瀏覽器**，前往博客來首頁：https://www.books.com.tw

2. **導航到您想要的分類頁面**：
   - 點擊首頁上的分類連結（例如「中文書 > 文學小說」）
   - 或使用網站頂部的分類選單

3. **複製完整 URL**：
   - 當您進入分類頁面後，從瀏覽器的網址列複製完整的 URL
   - 例如：`https://www.books.com.tw/web/sys_cebbotm/cebook/100307047/?o=1&v=1`

4. **在 config.py 中設定**：
   ```python
   BOOK_CATEGORIES = [
       ("中文書-文學小說", "https://www.books.com.tw/web/sys_cebbotm/cebook/100307047/?o=1&v=1", 100),
       # 添加更多分類 URL...
   ]
   ```

### 方法二：檢查網頁原始碼

1. **開啟分類頁面**後，按 `F12` 開啟開發者工具

2. **查看 Network 標籤**，重新載入頁面

3. **找到分類頁面的請求**，查看完整的 URL

### 方法三：使用搜尋功能（推薦）

**不需要提供分類 URL**，直接使用搜尋功能即可。程式會自動使用 `config.py` 中的 `SEARCH_KEYWORDS` 來搜尋書籍。

## 目前建議的做法

**使用搜尋功能**是最簡單且最穩定的方法：

1. 在 `config.py` 中的 `SEARCH_KEYWORDS` 列表添加您想要搜尋的關鍵字
2. 程式會自動使用這些關鍵字搜尋書籍
3. 不需要提供分類 URL

例如，如果您想收集程式設計相關的書籍，可以添加：
```python
SEARCH_KEYWORDS = [
    "Python",
    "Java",
    "JavaScript",
    "資料庫",
    "演算法",
    # ... 更多關鍵字
]
```

## 注意事項

- 分類 URL 可能會因為網站改版而失效
- 搜尋功能通常更穩定，因為它是網站的核心功能
- 如果分類瀏覽失敗，程式會自動切換到搜尋模式
- 建議主要依賴搜尋功能，分類瀏覽作為補充
